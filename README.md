Odin Recipes

This project is part of The Odin Project Foundations Course.
It is a simple website that showcases recipes using basic HTML.

What This Project Demonstrates

By completing this project, I will have practiced:

Creating and organizing a project directory

Building webpages using HTML elements

Linking between multiple pages

Using Git for version control (add, commit, push)

Cloning a repository using SSH

Writing meaningful commit messages

This project helps build fundamental skills in web development and Git workflow.
